# 🔐 ShieldScan // Web Security Dashboard

ShieldScan is a premium, interactive web-based security scanner dashboard. It audits URL hosts for essential security configurations, HTTP headers, open port services, and SSL/TLS handshake status.

## 🚀 Features

- **Dashboard Interface**: Stunning futuristic cyber layout with glassmorphic panels and live glowing charts.
- **Console Log Terminal**: Interactive command terminal simulator that reports live scan execution steps.
- **HTTP Header Audit**: Scans for standard headers like CSP, HSTS, X-Frame-Options, X-Content-Type-Options, etc.
- **Port Scanner Simulator**: Audits standard network service ports (SSH, HTTP, HTTPS, MySQL, etc.).
- **SSL/TLS Inspector**: Retrieves protocol versions, key lengths, signature details, and remaining cert validity.
- **Automatic Mitigation Engine**: Provides Nginx, Apache, and IIS code configurations to remediate found vulnerabilities.

## ⚙️ Technologies Used

- **HTML5**: Structured markup.
- **Vanilla CSS3**: Cyber theme design tokens, glassmorphism, responsive grids, and keyframe animations.
- **Vanilla JavaScript**: Scan orchestration engine, matrix rainfall rendering, interactive state management.
